AnalyticsType

public enum AnalyticsType : String

type defines the different types of extension analytics reports. overviewVersion1 returns analytics for the first 90 days, and overviewVersion2 returns analytics from January 31st, 2018 to the current day.

These reports are viewable and downloadable via a URL that is returned as a response.

A complete comparison is available here

  • overviewVersion1: The first version of extension analytics reports
  • overviewVersion2: The second version of extension analytics reports.
