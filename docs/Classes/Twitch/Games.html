Games

public struct Games

Games is a category of Twitch API calls that interacts with Games on Twitch. Despite the name, Games can refer to podcasts, art, or other streaming categories.

  • GetTopGamesResult

    GetTopGamesResult defines the different types of results that can be retrieved from the getTopGames call of the Games API. Variables are included that specify the data that was returned.

    • success: Defines that the call was successful. The output variable will contain all returned data.
    • failure: Defines that the call failed. Returns all data corresponding to the failed call. These data pieces are as follows:
    • Data? - The data that was returned by the API
    • URLResponse? - The response from the URL task
    • Error? - The error that was returned from the API call

  • getTopGames(tokenManager:after:before:first:completionHandler:)

    getTopGames will run the Get Top Games API call of the New Twitch API. This API call can be used to retrieve the top games being streamed on Twitch in a paginated fashion.

    This method does not require a TwitchTokenManager. No Authorization is required.

    More information about the web call is available here

    after

    The forward pagination token of the call.
    before

    The backwards pagination token of the call.
    first

    The number of objects to retrieve. Maximum 100. Default 20.
    completionHandler

    The function that should be run whenever the retrieval is successful. There are two types of GetTopGamesResult: success and failure.
  • getGames(tokenManager:gameIds:gameNames:completionHandler:)

    getGames will run the Get Games API call of the New Twitch API. This method can be used to get data regarding input game parameters. For instance, you can retrieve the ID of a game based on its name and vice-versa. A mix of IDs and names can be mixed to retrieve all data at once.

    This method does not require a TwitchTokenManager. No Authorization is required.

    More information about the web call is available here
