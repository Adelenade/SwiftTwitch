TwitchTokenManager

public class TwitchTokenManager

TwitchTokenManager is a class that is used to carry New Twitch API tokens.

If you wish to use a single token to pass between all API calls without having to keep reference to it, then simply set up the singleton TwitchTokenManager.

The singleton instance can be retrieved by using the following syntax: TwitchTokenManager.shared
